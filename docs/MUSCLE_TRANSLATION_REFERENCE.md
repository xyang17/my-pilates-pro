## 🎯 How to Use This Reference

### When Creating Exercises:

**Example 1: Pilates Push-up**
```
English: Core, Chest, Arms
Chinese: 核心, 胸部, 手臂
```

**Example 2: Single Leg Circle**
```
English: Hip Stabilizers, Core
Chinese: 臀部稳定肌肉, 核心
```

**Example 3: Leg Pull Front**
```
English: Core, Shoulders, Hamstrings
Chinese: 核心, 肩膀, 腘绳肌
```

---

## 💡 Notes for Your Use

1. **Be Consistent** - Always use the same English + Chinese pair for a muscle group
2. **Use Comma-Separated Lists** - `Core, Chest, Arms` not `Core/Chest/Arms`
3. **Title Case** - `Core` not `core`, `Hamstrings` not `hamstrings`
4. **Specific is Better** - Use `Hip Flexors` instead of just `Hips` when possible
5. **Most Common in Pilates** - You'll likely use: Core, Glutes, Hamstrings, Chest, Back, Shoulders, Arms, Spine
